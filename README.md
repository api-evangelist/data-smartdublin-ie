# Smart Dublin Open Data (data-smartdublin-ie)

Smart Dublin Open Data is a municipal government open-data portal for Ireland running **CKAN**, exposing a standardized catalog API over ~929 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-smartdublin-ie/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-smartdublin-ie-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.14

## API
- **Smart Dublin Open Data CKAN Action API** — base `https://data.smartdublin.ie/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.smartdublin.ie)
- Live endpoint verified: `https://data.smartdublin.ie/api/3/action/status_show` · ~929 datasets
